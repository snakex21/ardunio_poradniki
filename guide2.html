Wprowadzenie

Arduino to platforma open-source umożliwiająca tworzenie interaktywnych projektów elektronicznych. W tym poradniku poznasz podstawy działania i budowę płytek Arduino.

Przykładowy kod – Blikający LED

Poniższy kod sprawia, że wbudowana dioda LED miga co sekundę:


void setup() {
  pinMode(LED_BUILTIN, OUTPUT); // Ustawienie wbudowanej diody jako wyjścia
}

void loop() {
  digitalWrite(LED_BUILTIN, HIGH); // Włączenie LED
  delay(1000);                     // Czekaj 1 sekundę
  digitalWrite(LED_BUILTIN, LOW);  // Wyłączenie LED
  delay(1000);                     // Czekaj 1 sekundę
}

Szczegółowe wyjaśnienie

W funkcji setup() ustawiamy pin z diodą LED jako wyjście, a funkcja loop() w nieskończoność wykonuje polecenia włączania i wyłączania diody z 1-sekundowym opóźnieniem.
