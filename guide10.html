Poradnik 10: Projekt z wykorzystaniem wyświetlacza LCD

Powrót do strony głównej

Opis projektu

Wyświetlacze LCD umożliwiają prezentację danych. W tym poradniku poznasz, jak wyświetlić tekst na ekranie LCD przy użyciu biblioteki LiquidCrystal.

Przykładowy kod

Kod inicjalizuje wyświetlacz LCD (16x2) i wyświetla komunikat:


#include 

// Inicjalizacja: RS, E, D4, D5, D6, D7
LiquidCrystal lcd(12, 11, 5, 4, 3, 2);

void setup() {
  lcd.begin(16, 2); // Ustawienie wyświetlacza 16 kolumn x 2 wiersze
  lcd.print("Witaj w LCD!");
}

void loop() {
  // Dalsza obsługa może być dodana tutaj
}

Szczegółowe wyjaśnienie

Biblioteka LiquidCrystal upraszcza komunikację z wyświetlaczem LCD. Funkcja lcd.begin() ustawia rozmiar wyświetlacza, a lcd.print() wyświetla tekst.
