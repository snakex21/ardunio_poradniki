Poradnik 12: Monitorowanie parametrów środowiskowych

Powrót do strony głównej

Opis projektu

Dowiedz się, jak odczytywać temperaturę (np. za pomocą czujnika LM35) i przesyłać dane do monitora portu szeregowego, co pozwala na monitorowanie parametrów środowiskowych.

Przykładowy kod

Kod odczytuje napięcie z czujnika LM35 i przelicza je na temperaturę w stopniach Celsjusza:


const int sensorPin = A0;

void setup() {
  Serial.begin(9600);
}

void loop() {
  int sensorValue = analogRead(sensorPin);
  // Konwersja napięcia na temperaturę
  float voltage = sensorValue * (5.0 / 1023.0);
  float temperature = voltage * 100; // LM35 daje 10mV/°C
  Serial.print("Temperatura: ");
  Serial.print(temperature);
  Serial.println(" °C");
  delay(1000);
}

Szczegółowe wyjaśnienie

Odczyt analogowy z czujnika pozwala zmierzyć napięcie, które przeliczamy na temperaturę. W ten sposób możesz monitorować zmiany środowiskowe w czasie rzeczywistym.
