Poradnik 9: Obsługa przycisków i wyświetlanie wyników

Powrót do strony głównej

Opis projektu

Na tej stronie dowiesz się, jak podłączyć dwa przyciski i monitorować ich stan za pomocą Arduino, wysyłając wyniki do portu szeregowego.

Przykładowy kod

Kod poniżej odczytuje stany dwóch przycisków:


const int button1 = 2;
const int button2 = 3;

void setup() {
  pinMode(button1, INPUT_PULLUP);
  pinMode(button2, INPUT_PULLUP);
  Serial.begin(9600);
}

void loop() {
  int state1 = digitalRead(button1);
  int state2 = digitalRead(button2);

  Serial.print("Przycisk 1: ");
  Serial.print(state1 == LOW ? "Wciśnięty" : "Nie wciśnięty");
  Serial.print(" | Przycisk 2: ");
  Serial.println(state2 == LOW ? "Wciśnięty" : "Nie wciśnięty");
  
  delay(300);
}

Szczegółowe wyjaśnienie

Wykorzystanie funkcji digitalRead() pozwala na odczytanie stanu przycisków, a warunkowy operator (ternary) umożliwia wyświetlenie odpowiedniej informacji w monitorze portu szeregowego.
