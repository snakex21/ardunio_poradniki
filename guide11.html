Poradnik 11: Zdalne sterowanie przez Bluetooth

Powrót do strony głównej

Opis projektu

Bluetooth umożliwia bezprzewodową komunikację między Arduino a urządzeniami mobilnymi lub komputerami. Ten poradnik przedstawia podstawowy projekt sterowania modułem Bluetooth (np. HC-05).

Przykładowy kod

Kod wykorzystuje port szeregowy do komunikacji z modułem Bluetooth:


void setup() {
  Serial.begin(9600); // Komunikacja z modułem Bluetooth
  Serial.println("Moduł Bluetooth gotowy");
}

void loop() {
  if (Serial.available() > 0) {
    String command = Serial.readStringUntil('\n');
    Serial.print("Otrzymano: ");
    Serial.println(command);
    // Dodaj obsługę odebranych komend
  }
}

Szczegółowe wyjaśnienie

Moduł Bluetooth komunikuje się przez port szeregowy. Po odebraniu danych, Arduino odczytuje komendy, które mogą sterować różnymi elementami projektu.
