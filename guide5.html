Poradnik 5: Sterowanie diodą LED za pomocą przycisku

Powrót do strony głównej

Opis projektu

W tym poradniku dowiesz się, jak sterować diodą LED przy pomocy przycisku – gdy przycisk zostanie wciśnięty, LED się zapali.

Przykładowy kod

Kod odczytuje stan przycisku i w zależności od niego steruje diodą LED:


const int ledPin = 13;     // Pin diody LED
const int buttonPin = 2;   // Pin przycisku

void setup() {
  pinMode(ledPin, OUTPUT);
  pinMode(buttonPin, INPUT_PULLUP); // Użycie wewnętrznego rezystora podciągającego
  Serial.begin(9600);
}

void loop() {
  int buttonState = digitalRead(buttonPin);
  if (buttonState == LOW) { // Stan niski oznacza wciśnięcie przycisku
    digitalWrite(ledPin, HIGH);
    Serial.println("Przycisk wciśnięty - LED włączony");
  } else {
    digitalWrite(ledPin, LOW);
    Serial.println("Przycisk nie wciśnięty - LED wyłączony");
  }
  delay(200);
}

Szczegółowe wyjaśnienie

Przy użyciu digitalRead() odczytujemy stan przycisku. Wykorzystanie INPUT_PULLUP gwarantuje stabilny sygnał. W zależności od stanu, dioda LED jest włączana lub wyłączana.
