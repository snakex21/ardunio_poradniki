Poradnik 7: Sterowanie serwomechanizmem

Powrót do strony głównej

Opis projektu

Dowiedz się, jak sterować serwomechanizmem za pomocą Arduino, korzystając z biblioteki Servo.h.

Przykładowy kod

Kod poniżej steruje serwem, przesuwając je najpierw od 0 do 180 stopni, a następnie wracając:


#include 

Servo myServo;
const int servoPin = 9;

void setup() {
  myServo.attach(servoPin);
}

void loop() {
  // Obracanie serwa od 0 do 180 stopni
  for (int pos = 0; pos <= 180; pos++) {
    myServo.write(pos);
    delay(15);
  }
  // Obracanie serwa od 180 do 0 stopni
  for (int pos = 180; pos >= 0; pos--) {
    myServo.write(pos);
    delay(15);
  }
}

Szczegółowe wyjaśnienie

Korzystając z biblioteki Servo.h, dołączamy serwomechanizm do wybranego pinu. Pętle for odpowiadają za płynny ruch serwa w obu kierunkach.
