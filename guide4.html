Poradnik 4: Podstawy programowania w Arduino

Powrót do strony głównej

Podstawowe konstrukcje

Arduino wykorzystuje język oparty na C/C++. Poznasz tutaj zmienne, pętle i instrukcje warunkowe, niezbędne przy tworzeniu programów.

Przykładowy kod – Sterowanie warunkowe

Kod ilustruje użycie zmiennych, instrukcji warunkowych i pętli:


int liczba = 0;

void setup() {
  Serial.begin(9600);
}

void loop() {
  liczba++;
  if (liczba % 2 == 0) {
    Serial.println("Liczba parzysta");
  } else {
    Serial.println("Liczba nieparzysta");
  }
  delay(1000);
}

Szczegółowe wyjaśnienie

Zmienna liczba jest inkrementowana w każdej iteracji pętli. Instrukcja if sprawdza, czy liczba jest parzysta, a wynik jest wysyłany do monitora portu szeregowego.
