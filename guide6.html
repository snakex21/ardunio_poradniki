Poradnik 6: Odczyt danych z czujnika analogowego

Powrót do strony głównej

Opis projektu

Nauczysz się, jak odczytywać wartości z czujnika analogowego (np. potencjometru) i przesyłać je przez port szeregowy.

Przykładowy kod

Poniższy szkic odczytuje wartość z wejścia analogowego A0:


const int sensorPin = A0; // Wejście analogowe

void setup() {
  Serial.begin(9600);
}

void loop() {
  int sensorValue = analogRead(sensorPin);
  Serial.print("Wartość czujnika: ");
  Serial.println(sensorValue);
  delay(500);
}

Szczegółowe wyjaśnienie

Funkcja analogRead() pobiera wartość (0–1023) z czujnika. Dane te są następnie przesyłane przez port szeregowy, co umożliwia ich monitorowanie w czasie rzeczywistym.
